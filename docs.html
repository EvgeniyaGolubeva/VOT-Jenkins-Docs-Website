Setup Instructions

← Back to Home

1. Prepare Linux VM

Use a Linux machine. I used Ubuntu. Make sure it's updated:

sudo apt update && sudo apt upgrade

2. Install the tools

  • Git: sudo apt install git
  • Java 11+: required by Jenkins
  • Docker: for containerizing the site
  • Apache2: serves the site on port 80

3. Install Jenkins

Follow the official Jenkins docs or use:

wget -O /usr/share/keyrings/jenkins-keyring.asc https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key
sudo apt install jenkins

4. Create Jenkins Pipeline

I created a pipeline that:

  1. Clones the GitHub repo
  2. Builds a Docker image (optional)
  3. Deploys HTML to /var/www/html

5. GitHub Webhook

I used ngrok to expose Jenkins to GitHub so it can trigger builds on push. Add the public ngrok URL as a webhook in your GitHub repo settings.

Done!

Now every push updates the live website automatically!
