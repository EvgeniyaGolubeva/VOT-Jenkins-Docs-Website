Jenkins Deployment

A project with CI/CD by Jenkins, Apache, Docker and Github

By Evgeniya Golubeva from 11V TUES

Read the Docs

What is this?

This is a simple HTML site deployed automatically with Jenkins. Every time you push to GitHub, Jenkins pulls the code, builds a Docker container, and deploys it to Apache.

Tech used

  • ✔️ Jenkins (for automation)
  • ✔️ GitHub + Webhook
  • ✔️ Apache Server
  • ✔️ Docker
  • ✔️ HTML & CSS
